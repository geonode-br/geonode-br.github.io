IDE livre e de Código Aberto

GeoNode é um aplicativo baseado na web e plataforma para desenvolvimento de sistemas de informação geoespacial (GIS) e para implantação de infraestruturas de dados espaciais (IDE). O Geonode oferece suporte uma gama de padrões OGC & ISO/TC 211

Conheça o Projeto »

Extensível e Interoperável

O GeoNode é projetado para ser estendido e modificado e pode ser integrado a plataformas existentes. É construído sobre uma plataforma de componentes de código aberto comprovados, incluindo Python/Django, GeoServer, pycsw, OpenLayers e outros.

Para desenvolvedores »

Gerenciamento Colaborativo

Os componentes do GeoNode facilitam a interação com outras implantações do GeoNode, serviços OGC externos, bem como outros serviços e ferramentas sociais populares - Com foco em interoperabilidade e distribuição de dados abertos
